# Big Data Course Project

大数据原理与实践课程项目

## Project Structure

```
bigdata-backend/     # Spring Boot 后端
cloud-storage-frontend/  # Vue.js 前端
```

## Tech Stack

- **Backend**: Spring Boot, MyBatis, MySQL
- **Frontend**: Vue.js, Element UI

## Quick Start

### Backend

```bash
cd bigdata-backend/bigdata后端集成版本/bigdata
mvn spring-boot:run
```

### Frontend

```bash
cd cloud-storage-frontend/cloud-storage-frontend
npm install
npm run serve
```
